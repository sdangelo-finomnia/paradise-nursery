# Paradise Nursery

Benvenuti in Paradise Nursery, la tua destinazione ideale per piante d'interni! Questa è un'applicazione e-commerce React che ti consente di esplorare una vasta gamma di piante, aggiungerle al carrello e gestire facilmente gli articoli.

**Caratteristiche:**
- Naviga tra una varietà di piante
- Aggiungi piante al carrello con un semplice clic
- Gestisci gli articoli nel tuo carrello

Costruito con React e Redux per una gestione dello stato efficiente.
